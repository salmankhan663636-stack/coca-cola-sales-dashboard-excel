Coca-Cola USA Sales Dashboard (Excel)
Project Overview

This project is an interactive Microsoft Excel dashboard developed to analyze Coca-Cola USA retail sales performance.

The dashboard provides key business insights through KPI cards, Pivot Tables, Pivot Charts, and Slicers, enabling users to monitor sales trends and compare performance across multiple dimensions.

Key Features:
Interactive Region Slicer
Invoice Year Filter
KPI Cards
Total Sales
Operating Profit
Units Sold
Average Price per Unit
Sales by Brand Analysis
Sales by Retailer Analysis
Quarterly Sales Trend
Operating Profit Margin Analysis




Business Questions Answered:
Which brand generated the highest sales?
Which retailer contributed the most revenue?
How did sales change between 2022 and 2023?
Which region performed the best?
What is the quarterly sales trend?
How did operating profit margin change over time?
